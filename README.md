# guia-estudos-webdev-notebooklm
Desenvolvimento Web: Fundamentos e Práticas Essenciais Este repositório contém o material desenvolvido para o desafio da DIO utilizando o NotebookLM. O objetivo é sistematizar o conhecimento sobre a tríade fundamental da web (HTML, CSS, JS) e arquitetura cliente-servidor.
## Analogia: O Site como uma Lanchonete

Para entender como um site funciona, imagine que você está abrindo uma lanchonete:

*   **HTML (A Estrutura):** É a planta da lanchonete, as paredes, o balcão e onde ficam as mesas. Sem ele, não há lugar para o negócio existir.
*   **CSS (A Decoração):** É a pintura, a logomarca, as cores das cadeiras e o design do cardápio. É o que faz o cliente sentir vontade de entrar.
*   **JavaScript (O Atendimento):** É o garçom e o sistema de pedidos. Ele anota o que você quer, processa o pagamento e traz o lanche. É a parte que "funciona" e interage com você.

---

## 📚 Glossário Técnico

*   **HTML:** Linguagem de marcação usada para estruturar os elementos da página.
*   **CSS:** Linguagem de estilos usada para definir a aparência visual.
*   **JavaScript:** Linguagem de programação que permite criar conteúdo dinâmico e interativo.
*   **API:** Um conjunto de regras que permite que dois sistemas "conversem" entre si.
*   **Servidor:** O computador remoto que guarda os arquivos do site e os envia quando alguém faz um pedido.
*   **Cliente:** O seu navegador (como o Opera GX) que recebe os arquivos e os mostra para você.

## Engenharia de Prompts e "Cicatrizes"

Nesta seção, documento os desafios encontrados ao utilizar a IA para gerar este guia:

1. **Bug de Formatação (".O."):**
   - **Problema:** A IA repetia o caractere ".O." ao final de todas as frases.
   - **Causa Raiz:** Conflito entre a tradução automática do navegador Opera GX e a resposta nativa da IA em português.
   - **Solução:** Desativação do tradutor do navegador, o que normalizou a escrita.

2. **Refinamento de Analogia:**
   - **Problema:** A IA tentou usar uma analogia de "construção de casa" que ficou confusa e com erros de sintaxe.
   - **Solução:** Intervi com um novo prompt solicitando a analogia da "Lanchonete", que é mais didática para o público iniciante.

3. **Curadoria de Conteúdo:**
   - **Problema:** As primeiras respostas eram muito longas e técnicas.
   - **Solução:** Utilizei prompts de restrição ("seja direto", "evite textos longos") para garantir que o guia fosse fácil de ler.
